Autonomous Facial Recognition Quadcopter

3D Printed Material


Our product's frame work was created with an original design. Using two 50 cm carbonfiber tubes to create the general shape, we had to 3D print multiple components.


Hardware


As mentioned in our about us, our main hardware pieces are the STM32F407 Discovery Board and the Raspberry Pi 4. Aside from these two components, we utilized an IMU that uses a BNO055 sensor by Bosch Sensortec. To help connect all of these components, we also designed an interposer board.
